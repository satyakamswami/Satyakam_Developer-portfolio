# Satyakam Dev

A cinematic luxury developer portfolio website designed and developed by Satyakam Swami.

Built using pure HTML and CSS, this project combines fullscreen video backgrounds, responsive layouts, dark futuristic aesthetics, and smooth UI interactions to create a modern premium web experience.
